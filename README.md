
# Get IP Find 🔍

A fast and efficient tool to retrieve detailed information about any IP address. This tool is built using a combination of **Python** and **Bash** for maximum compatibility in terminal environments.

## 🚀 Features
* **Detailed Information:** Get Country, Region, City, ISP, and more.
* **Dual Scripting:** Includes both Python (.py) and Bash (.sh) versions.
* **Lightweight:** Designed specifically for Termux and Linux users.

## 🛠 Installation & Usage

To use this tool in Termux, follow these steps:

1. **Update Packages:**
   \`\`\`bash
   pkg update && pkg upgrade
   \`\`\`

2. **Clone the Repository:**
   \`\`\`bash
   git clone https://github.com/zxmoon76-sys/get-ip-find
   \`\`\`

3. **Run the Tool:**
   - **Using Python:**
     \`\`\`bash
     python get_ip.py
     \`\`\`
   - **Using Bash:**
     \`\`\`bash
     bash get_ip.sh
     \`\`\`
